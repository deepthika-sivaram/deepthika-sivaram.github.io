# Deepthika Sivaram — Portfolio 🚀

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB) ![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white) ![Sass](https://img.shields.io/badge/Sass-CC6699?style=for-the-badge&logo=sass&logoColor=white) ![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-222222?style=for-the-badge&logo=github&logoColor=white)

Personal portfolio website built with React + TypeScript, deployed on GitHub Pages.

🌐 **Live site:** [deepthika-sivaram.github.io](https://deepthika-sivaram.github.io)

## Features

- Dark & light mode toggle
- Responsive & mobile-friendly
- Photo carousel in About section
- Interactive contact form (EmailJS)
- Resume viewer & download
- Project showcase & timeline

## Tech Stack

- React 18 + TypeScript
- SCSS for styling
- Material UI components
- EmailJS for contact form
- GitHub Actions for CI/CD

## Deployment

Deployed automatically via GitHub Actions on every push to `main`.
